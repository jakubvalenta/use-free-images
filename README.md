# Use Free Images

Bulk download URL and copyright information about images from Flickr, FreeImages.com and MorgueFile.

## Installation

Install with pip

```
pip install use-free-images
```

or with setuptools.

```
python setup.py install
```

## Usage and help

```
use-free-images -h
```
